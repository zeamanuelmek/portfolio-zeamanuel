# Design Principles

The SemEnaWerk design system draws from Ethiopian cultural heritage to create interfaces that feel both
familiar and modern. Every design decision serves a purpose: rooted in culture, optimized for usability.

---

## Ethiopian Cultural Elements

### Gold (#D4AF37): Heritage & Craftsmanship

<div style={{ borderLeft: '4px solid #D4AF37', paddingLeft: '16px', margin: '16px 0' }}>
Ethiopian gold represents centuries of craftsmanship and quality. In our system, gold is the **primary
action color**: it signals "this is important, this is valuable." Used for primary buttons, active
states, and premium features.
</div>

Gold appears as:
- **Solid fills**: primary buttons, active badges
- **Gradients**: headings with `goldGradient` prop, decorative elements
- **Glow effects**: hover states on cards via `goldGlow` prop
- **Borders**: focus rings, selected states

### Habesha Patterns: Textile Heritage

The traditional woven patterns of Ethiopian textiles (habesha kemis) inspire our decorative elements:

- **Habesha Divider**: triple-line divider in green, gold, and red (Ethiopian flag colors)
- **Diamond Divider**: geometric motif inspired by textile weaving patterns
- **Pattern backgrounds**: subtle geometric fills for section backgrounds

These elements are available through the `Divider` component's `habesha` and `diamond` props.

## Interaction Patterns

### Gold Glow on Hover

Primary interactive elements receive a **gold box-shadow lift** on hover:

```css
box-shadow: 0 4px 20px rgba(212, 175, 55, 0.3);
transform: translateY(-2px);
```

This creates a "floating" effect that draws attention to interactive elements. Applied via the `goldGlow`
prop on the Card component — activates on hover automatically.

**Why gold glow?** It creates a premium, tactile feel. Users perceive gold-glowing elements as more
valuable and interactive, increasing engagement with primary actions.

### Glass Morphism

Navigation and overlay elements use subtle transparency:

```css
background: rgba(255, 255, 255, 0.8);
backdrop-filter: blur(10px);
```

This creates depth without heaviness — modern interfaces that feel light and spacious.

### Generous Border Radius

Components default to `radius="lg"` (buttons, inputs) and `radius="xl"` (cards). This creates a
friendly, approachable aesthetic that contrasts with sharp corporate designs.

## Typography Philosophy

### Three-Font System

| Font | Role | Character |
|------|------|-----------|
| **Space Grotesk** | Headings | Geometric, modern, distinctive |
| **Inter** | Body text | Highly readable, neutral, professional |
| **JetBrains Mono** | Code/data | Technical, precise, monospaced |

Space Grotesk was chosen for headings because its geometric forms echo the angular patterns in
Ethiopian art and architecture, while remaining thoroughly modern.

### Scale & Hierarchy

Headings use a deliberate size scale that creates clear visual hierarchy:
- `h1: 2.5rem` — page titles, hero text
- `h2: 2rem` — section headers
- `h3: 1.5rem` — subsection headers
- `h4: 1.25rem` — card titles, feature names

## Component Philosophy

### Mantine Base + Ethiopian Brand Layer

Every component starts with a Mantine primitive and adds an Ethiopian design layer:

```
Mantine Button → gold color default + lg radius → SemEnaWerk Button
Mantine Card   → goldGlow hover + xl radius    → SemEnaWerk Card
Mantine Alert  → semantic color mapping         → SemEnaWerk Alert
```

**Why Mantine?** It provides accessible, tested, production-ready components. We add brand identity
on top — not reinventing the wheel, but painting it gold.

### Single Responsibility

Each atom does **one thing well**:
- `Button` handles clicks — never layout
- `Card` contains content — never navigation
- `Container` constrains width — never decorates

This makes components predictable, composable, and easy to test.

### Theme-Driven, Not Hardcoded

Every color, spacing value, and shadow comes from the theme system:

```typescript
// Good — uses theme tokens
<Button color="gold">Pay with TeleBirr</Button>
<Card style={{ boxShadow: theme.other.goldGlow }}>

// Bad — hardcoded values
<Button style={{ background: '#D4AF37' }}>
<Card style={{ boxShadow: '0 4px 20px rgba(212,175,55,0.3)' }}>
```

This enables **5-minute theme swaps** — change one file, entire site recolors.

## Accessibility

### Commitments

- **Color contrast** — all text meets WCAG 2.1 AA (4.5:1 for body, 3:1 for large text)
- **Focus indicators** — visible gold focus rings on all interactive elements
- **Screen readers** — semantic HTML, ARIA labels on icon-only buttons
- **Keyboard navigation** — all components are fully keyboard accessible via Mantine
- **Motion** — respects `prefers-reduced-motion` for animations

### Ethiopian Context

- **Mobile-first** — the Ethiopian market is mobile-dominant; every component is designed for touch
- **Low bandwidth** — minimal JavaScript, optimized assets, no heavy animations
- **Multilingual-ready** — components handle Amharic text (right-to-left not needed, but long characters supported)
